Sistema de Administración Escolar

Un Sistema de Administración Escolar Integrado para mejorar la gestión académica.

La API proporciona acceso a datos relacionados con los estudiantes, permitiendo la definición de perfiles personales y la gestión de documentos importantes, como actas de nacimiento, CURP y comprobantes de domicilio.

Creado usando Spring Boot en Java como framework y PostgreSQL como base de datos.
